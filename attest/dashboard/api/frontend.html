Four steps to your first evaluated test run. Each card jumps to the right page.

🛡️ What is ATTEST?

ATTEST (Agent Testing & Trust Evaluation Suite) is an end-to-end framework for testing AI agents. It combines fast, free deterministic assertions (structure, latency, cost, PII, tool calls) with LLM-based evaluators (relevancy, faithfulness, safety, tone, and more) so you can measure both correctness and trust in a single run — locally or in CI.

What it can do

🔌
9 agent adapters
HTTP, Callable, Foundry, LangChain, LangGraph, CrewAI, AutoGen, OpenAI Assistants & MCP — plus an offline mock adapter that powers the bundled demo.
✅
34 assertions
Deterministic checks — JSON shape, latency, TTFT, throughput, cost, no-PII, language, semantic match & more.
🧪
36 LLM evaluators
Built-in + DeepEval + Azure AI evaluators, plus RAGAS for retrieval quality.
💬
Multi-turn & simulation
Single-turn, scripted conversations, and goal-driven user simulations.
🔀
Multi-agent routing
Test orchestrators that delegate to sub-agents — verify the routing path & which agent handled each request.
📸
Baselines & regression
Snapshot golden responses and fail the build when behavior drifts.
📊
Reports & CI
HTML / Markdown / JUnit reports, run comparison, and CI templates.

Key concepts

Assertions vs. Evaluators
Assertions are instant, free, pass/fail checks. Evaluators call an LLM to score qualities like relevancy or safety (0–1). To save cost, evaluators are skipped when assertions fail.
Suites & Tags
File suites group tests by YAML file. Tag suites are virtual groups across files — run either as a unit from the Run page.
Runs & Comparison
Every execution is saved to run history. Name runs and compare any two side-by-side to see score improvements or regressions.
Multi-agent routing
For orchestrators that delegate to sub-agents, point an HTTP agent's handled_by / routing_path fields at its response (in Agent Setup), then assert delegation with routed_to, not_routed_to, routing_path and routing_path_contains. In-process graphs (LangGraph, CrewAI, AutoGen) test the same way via their adapters.
